# 🧠 Quiz: Empirismo vs Racionalismo

Projeto desenvolvido em JavaScript para um trabalho escolar com o objetivo de explorar conceitos filosóficos de forma interativa.

O usuário responde perguntas e descobre se possui um perfil mais empirista ou racionalista.

---

## 🎯 Funcionalidades

- Início do quiz com botão interativo
- Perguntas e respostas dinâmicas
- Exibição de resultado final
- Imagem temática do quiz
- Link de apoio para material em PDF

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## ▶️ Como executar o projeto

1. Baixe ou clone este repositório
2. Abra o arquivo `index.html` no navegador
   
Ou também pode acessar o projeto online:

👉 https://anaperessim.github.io/filosofia-quiz/

## 📚 Contexto do projeto

Este quiz foi criado como parte de um trabalho escolar para aplicar conceitos de lógica de programação e manipulação do DOM com JavaScript.
